== Description ==

attachement-widget est un plugin qui permet d'afficher, dans un widget, les pi&egrave;ces jointes &agrave; un article ou une page (sauf les images) sous forme de liste avec l'icone correspondant au type de la pi&egrave;ce jointe.

== Installation ==

1. T&eacute;l&eacute;chargez `attachement-widget.php` vers le r&eacute;pertoire `/wp-content/plugins/attachement-widget/`
1. Activez l'extension &agrave; travers le menu 'Extensions" dans WordPress
1. Allez dans le menu 'Widget' pour le glisser dans une zone de widgets et le param&eacute;trez
1. Vous pouvez param&eacute;trer le titre du widget, en une ou deux lignes
